  자기소개 


           안녕하십니까!
           저는 2018학년도에 성신여자대학교 공과대학에 입학한 22살 이혜인입니다. 저의 주전공은 정보시스템공학과로서 IT계열에 속합니다.
           저는 입학후에 제 진로에 관련하여 대외활동 부분에서 많은 생각을 하게되었고, 그 부분에 관해서 알아보다가 '멋쟁이 사자처럼'이라는 동아리를 발견하게 되었습니다.
           1학년 때 강의시간에 C++과 HTML, CSS, JAVA script 를 배웠었는데 그 때 저는 웹 프로그래밍 쪽으로 흥미를 느꼈습니다.
           또한 이번학기부터 저는 학교에서 ‘서비스디자인공학과’ 를 부전공으로 선택하여 공부하고 있는데, 포토샵과 일러스트등 디자인 관련한 프로그램을 심도있게 다루기 때문에
           1학년 때 저의 결과물보다 더 향상된 웹페이지를 만들 수 있는 능력을 동아리와 연계하여 기르고 싶습니다.


  내가 만들고 싶은 서비스


          웹 서비스 

           저는 ‘ 화면 분할 웹 사이트’ 를 고안했습니다.
           수강신청을 하거나, 콘서트를 가거나 할 때 핵심 사용 사이트 이외에도 시간을 초단위로 나타내주는 부가적인 사이트를 이용할 때가 있습니다.
           비슷한 예시로, 레포트를 작성 할 때 사전이나 논문을 보여주는 사이트와 작성하는 워드 프로그램을 같이 사용할 때가 있습니다.
           사용자가 프로그램 창을 두 개 이상 사용할 경우에 물론 임의로 크기를 조정해서 사용할 수 있지만, 마우스를 이용하여 창을 옮기고 줄이고 늘리고 하는 과정이 불필요하다고 생각하였고
           실제로 제가 앞서 말한 방법을 사용할 때에 불편함을 느꼈기 때문에 충분히 더 편리하게 창의 크기를 변경할 수 있는 프로그램이 개발될 수 있을 것 같았습니다.
           따라서 ‘드래그’ 방식을 이용하여 사용자가 이용 할 사이트를 웹사이트에 넣으면 사용자가 원하는 비율을 입력받아서
           사이트 두 개를 편리한 방법으로 동시에 사용할 수 있도록 하는 웹사이트가 있으면 좋을 것 같다고 생각하였습니다.

          앱 서비스 

          저는 ‘달걀 조리 타이머 앱’ 을 고안하였습니다.
          저는 현재 학교 근처에서 자취를 하고 있는데, 자취생의 특성상 비싼 재료를 사용하거나 요리과정이 복잡한 요리를 해먹기에는 불편함이 있어서
          달걀이나 통조림 같은 주변에서 쉽게 접할 수 있는 단순한 재료로 주 끼니를 해결하고 있습니다.
          그러던 와중에 반숙과 완숙 같은 달걀 상태에 따른 삶는 시간이 정해져 있다는 정보를 찾게되었고, 그 시간에 따라 타이머를 맞추어서 달걀을 삶기 시작하였습니다.
          하지만 삶는 시간은 단순한 정보이기 때문에 하루 삶고 나면 다음날 까먹기 일쑤이고, 때마다 달걀 상태를 검색하고 타이머로 시간을 다르게 맞추는 것이 번거롭다고 느껴졌습니다.
          만약 달걀 상태를 선택하여 자동으로 타이머가 맞춰지는 앱이 있다면 자취생과 같은 1인 가구를 상대로 많은 도움이 되는 어플이 될 것 같습니다.
          나아가서 달걀을 삶는데에 그치지 않고 달걀 찜이나 수란 , 머랭과 같은 달걀과 관련된 요리에서도 조리시간에 따른 타이머를 구현하면 편리하게 쓰일 것 같습니다.
